# 个人网站项目 — CLAUDE.md

## 项目简介

这是一个 HR 专业人士的个人展示网站，用于向外部访客（猎头、合作方、求职者等）展示个人简历与相关文档资料。
网站风格专业、简洁，内容清晰易读，无需用户登录或复杂交互。

---

## 技术栈

- **HTML5 + CSS3 + 原生 JavaScript**（无框架，保持简单可维护）
- 字体：Google Fonts（推荐 Inter 或 Noto Sans SC，支持中文）
- 部署：GitHub Pages 或 Vercel（静态托管，免费）
- 不引入任何后端或数据库

---

## 目录结构

```
/
├── index.html          # 主页（About Me + 导航）
├── resume.html         # 简历页
├── docs.html           # 文档展示页（可下载或在线预览）
├── style.css           # 全局样式
├── main.js             # 简单交互逻辑（如平滑滚动）
└── assets/
    ├── docs/           # 存放 PDF 等文档文件
    └── images/         # 头像或背景图
```

---

## 页面内容规划

### index.html — 首页
- 顶部导航栏（姓名 logo + 页面跳转链接）
- Hero 区块：姓名、职位头衔、一句话自我介绍
- 快速入口卡片：跳转到简历页 / 文档页

### resume.html — 简历页
- 工作经历（公司、职位、时间、职责描述）
- 教育背景
- 技能标签
- 提供简历 PDF 下载按钮

### docs.html — 文档展示页
- 文档列表：文件名、描述、上传日期
- 每份文档提供「在线预览（iframe 嵌入 PDF）」和「下载」两个按钮
- 可按类别分组展示（如：证书、作品、报告）

---

## 设计规范

- **配色**：白色背景 + 深灰文字，主色调用一个低饱和度的蓝色或绿色作为点缀
- **风格**：专业、干净，避免花哨动画
- **字号**：正文 16px，标题层级清晰（H1 > H2 > H3）
- **响应式**：必须在手机端正常显示（移动端优先）
- **无障碍**：图片加 alt 属性，对比度符合 WCAG AA 标准

---

## 编码规范

- 所有注释和变量命名使用英文
- CSS 使用 BEM 命名规范（如 `.card__title`, `.nav__link--active`）
- JavaScript 保持最小化，只做必要的 DOM 操作
- 不使用任何 CSS 框架（如 Bootstrap、Tailwind），手写样式以便完全掌控

---

## 文档处理方式

- PDF 文件放在 `assets/docs/` 目录下
- 在线预览使用 `<iframe src="assets/docs/xxx.pdf">` 实现
- 若文件较大，提供直接下载链接即可，不强制预览
- 文件命名规范：`YYYYMM_描述.pdf`，例如 `202503_resume.pdf`

---

## 注意事项

- 网站不收集任何用户数据，不接入分析工具（除非用户主动要求）
- 联系方式仅展示 LinkedIn 或邮箱，不放手机号
- 所有文档在上传前确认内容不含敏感个人信息
- 每次修改后在本地用浏览器测试手机和桌面两种视图再提交
