# Documents Directory

This directory should contain your PDF files for the portfolio website.

## File Naming Convention

Use the format: `YYYYMM_description.pdf`

Example: `202503_resume.pdf` for a resume updated in March 2025

## Notes

- Replace these placeholder files with your actual documents
- Ensure PDF files are optimized for web (file size under 5MB recommended)
- The preview functionality works best with PDFs that allow embedding
- If a PDF is too large, consider providing only the download option
